Requerimientos en una base de datos para un modelo de clasificacion:

Toda base de datos para un modelo de Clasificacion debe cumplir las siguientes condiciones:¶

- Al menos 500 registros
- Los datos no deben presentar valores nulos o vacios ni cambios de escala de medicion o formato
- La base de datos debe contar con una variable de etiquetas o clases a clasificar bien definida
- Ninguna de las clases debe ser equivalente a mas del 60% de los registros de la base
- Debe existir una relacion entre las variables explicativas y la variable a predecir

Para este ejercicio se utiliza la base de datos libre de kaggle "mobile Price Classification" accequible desde el link https://www.kaggle.com/iabhishekofficial/mobile-price-classification.¶

El contexto del ejercicio se basa en predecir a que rango de precios pertenese un telefono mobil segun sus componentes, capacidad en memoria, ram y demas caracteristicas tecnicas:

Para este ejercicio se realiza una transformación matemática en la que las 20 variables explicativas iniciales se convierten en 2 variables "componente principal" y se grafica en un plano. Esta técnica es llamada PCA y si bien no permite un alto grado de confianza en la diferenciación, es una excelente base descriptiva para empezar a abordar el problema y confirmar que existen relaciones lineales y no lineales que permitan la diferenciación.¶
